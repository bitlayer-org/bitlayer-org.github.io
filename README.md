# Bitlayer docs

## Edit
```bash
git clone git@github.com:bitlayer-org/bitlayer-org.github.io.git 
cd bitlayer-org.github.io
git checkout -b main origin/main

# edit docs locally
```

## Preview locally
just skip this step if preview locally is not needed. 

```bash
yarn start
```

## Publish
```bash
cd bitlayer-org.github.io
git add .
git commit -m "update doc"
git push

# auto publish
```
